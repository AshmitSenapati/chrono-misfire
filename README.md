# Chrono Misfire (Twisted Mechanic Game)

## Core Mechanic
Shooting targets.

## Twist (Core Rule Broken)
Time speed changes based on accuracy:
- ✔ Correct hit → slows/stabilizes time
- ✖ Wrong hit / Miss → speeds up time
- If time reaches 4.0x → you lose

This twist is central: removing it breaks the entire challenge and difficulty system.

## Controls
- Mouse Move: Aim
- Left Click: Shoot
- R: Restart

## Win / Lose
- Win: Survive 60 seconds
- Lose: Time multiplier reaches 4.0x

## Platform
Web / HTML5 (Phaser.js)
